# User interface

## Timeline view

The timeline view displays your media library with items grouped by day. It is possible to search and filter by rating via the UI. The view also accepts query params for filtering that are not yet available via the UI.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/rgallery-timeline-3.png" alt="rgallery timeline" width="3000" height="1875" srcset="https://packages.rmilo.dev/images/v0.0.4/rgallery-timeline-3-800.png 800w, https://packages.rmilo.dev/images/v0.0.4/rgallery-timeline-3-1600.png 1600w" >}}

## Memories

If media items exist "on this day" from year(s) ago, they will be displayed at the top of the timeline. The memories UI can be dismissed for the day, which is persisted via the browser's local storage.

Click the "X days ago" to scroll to that date on the timeline.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-memories.png" alt="rgallery memories" width="2880" height="722" srcset="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-memories-800.png 800w, https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-memories-1600.png 1600w" >}}

## Favorites

The favorites view shows all media items with an EXIF rating of 5. Favorites can also be viewed in the timeline view by using the star filter.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-favorites.png" alt="rgallery favorites" width="3000" height="1875" srcset="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-favorites-800.png 800w, https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-favorites-1600.png 1600w" >}}

## Folders

The folders view shows a navigable list of all folders found in the media directory.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-folders.png" alt="rgallery folders" width="3000" height="1875" srcset="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-folders.png 2357w" >}}

## Tags

The tags view shows a navigable list of all EXIF tags. The EXIF field "subject" is used.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-tags.png" alt="rgallery tags" width="3000" height="1875" srcset="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-tags-800.png 800w, https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-tags-1600.png 1600w" >}}

## Gear

The gear view shows a navigable list of cameras, lenses, focal lengths, and more along with the total count of media items.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-gear.png" alt="rgallery gear" width="3000" height="1875" srcset="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-gear-800.png 800w, https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-gear-1600.png 1600w" >}}

## Map

The map view shows a map of all media items that contain GPS coordinates.

{{< figure src="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-map.png" alt="rgallery map" width="3000" height="1875" srcset="https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-map-800.png 800w, https://packages.rmilo.dev/images/v0.0.4/ui/rgallery-map-1600.png 1600w" >}}
